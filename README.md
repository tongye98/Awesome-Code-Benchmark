## Taxonomy

This list organizes code benchmarks by primary capability and software-engineering workflow. Each benchmark entry includes compact metadata for task type, granularity, interaction pattern, and evaluation method.

| Dimension | Values |
|:--|:--|
| Granularity | Function, API, Query / Database, File, Project, UI, Repository, Workflow |
| Interaction | Single-turn, Multi-turn, Agentic, Async, Multi-agent |
| Evaluation | Unit Tests, Execution, Performance, Human, LLM-as-Judge, Security Exploit, Economic |
| Environment | None, Sandbox, Terminal, Browser, IDE, CI |
| Freshness | Static, Dynamic, Held-out, Contamination-resistant |
